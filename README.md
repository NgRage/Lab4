# Фролов Дмитрий КМБ Лабораторная №4

# Задание 1

## Задача 1

### Текст задачи
Дерево содержит строки. Заменить в каждой строке последний символ на заданный.


### Алгоритм решения
1) Запросить у пользователя N + проверка
2) Сгенерировать дерево строк с N элементами + показать промежуточный результат
3) Запросить у пользователя один символ + проверка
4) При помощи функции на основе map заменить в каждой строке последний символ на заданный
5) Показать финальный результат
